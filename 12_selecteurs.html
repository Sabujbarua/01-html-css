  1. font-weight: La propriété CSS font-weight permet de définir la graisse utilisée pour le texte.
  2. letter-spacing: La propriété letter-spacing définit l'interlettre utilisée pour les caractères qui composent le texte.
  3. word-spacing: La propriété word-spacing définit la règle d'espacement utilisée entre les balises et les mots.
  4. text-shadow: La propriété text-shadow ajoute des ombres au texte.
  5. background-size: La propriété CSS background-size définit la taille des images d'arrière-plan pour l'élément.
  6. opacity: La propriété opacity définit la transparence d'un élément. Autrement dit, elle permet de définir le degré de visibilité de l'arrière-plan sur lequel est placé l'élément.
  7. border-style: La propriété border-style est une propriété raccourcie qui permet de définir le style des lignes utilisées pour les bordures des quatre côtés de la boîte d'un élément.
  8. box-shadow: La propriété box-shadow décrit les ombres appliquées à la boîte avec une liste d'ombres séparées par des virgules
  9. list-style-image: La propriété list-style-image définit l'image utilisée comme puce devant les éléments de listes. On peut également utiliser la propriété raccourcie list-style.
  10. border: La propriété border est une propriété raccourcie qui permet de définir les propriétés liées à la bordure.
  11. border-radius: La propriété border-radius permet de définir des coins arrondis pour la bordure d'un élément.
  12. border-width: La propriété border-width définit la largeur d'une bordure pour la boîte d'un élément.
  13. font-variant: La propriété font-variant est une propriété raccourcie représentant les propriétés : font-variant-caps, font-variant-numeric, font-variant-alternates, font-variant-ligatures et font-variant-east-asian. On peut aussi utiliser les valeurs de la propriété font-variant définies avec CSS Level 2 (normal ou small-caps), en utilisant la propriété raccourcie font.
  14. background-color: La propriété background-color permet de définir la couleur utilisée pour l'arrière-plan d'un élément (celle-ci peut être une couleur transparente).
  15. background-image: La propriété background-image permet de définir une ou plusieurs images comme arrière(s)-plan(s) pour un élément.
  16. !important: Sous la pression d'un Consortium issu des principaux frameworks tels que Bootstrap, Foundation et Angular, le W3C s'est récemment penché sur la célèbre fonction CSS !important et propose dorénavant deux innovations qui en étendent l'efficacité et qui vont se révéler bien pratiques : !veryimportant et !notveryimportant.
